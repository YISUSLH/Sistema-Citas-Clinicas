# Sistema de Administración de Citas para un Consultorio Clínico

## Instalación y configuración

Para ejecutar este proyecto es necesario contar con:

- Java Development Kit 11.
- IntelliJ IDEA.
- Git.
- Una cuenta de GitHub.

El proyecto fue creado en Java y utiliza Git para el control de versiones.

## Uso del programa

El programa permitirá administrar citas de un consultorio clínico.  
Sus funciones principales serán:

- Registrar doctores.
- Registrar pacientes.
- Crear citas con fecha y hora.
- Relacionar una cita con un doctor y un paciente.
- Permitir el acceso únicamente a administradores autorizados.
